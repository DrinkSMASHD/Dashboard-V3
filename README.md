# SMASHD Dashboard V3

Working Vite dashboard for Vercel.

## Deploy

Upload the CONTENTS of this folder to the GitHub repo root.

In Vercel:
- Framework Preset: Vite
- Build Command: npm run build
- Output Directory: dist

## Features

- Inline update buttons
- Dynamic totals recalculated instantly
- Data saved in browser localStorage
- SMASHD logo branding
- Bill/Benny/Tim/Andy avatars
- Progress bars use #9eee90
